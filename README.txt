
KemDash Website Prototype
-------------------------
This is a fully static prototype built for KemDash (Voice Over) 
- Dark theme with black->orange gradient
- Arabic + English toggle (client-side)
- Pages: index.html, about.html, services.html, portfolio.html, contact.html
- Placeholders for logo, audio and portfolio thumbnails are in assets/img and assets/audio
Replace assets with real content (logo.svg, audio sample at assets/audio/sample.mp3, thumbs) and upload to any static host (Netlify, GitHub Pages, Vercel, etc.).
